БЫСТРЫЙ ЗАПУСК (Win):

1. Убедитесь, что у вас установлен Python 3.8+
   - Скачайте с https://python.org
   - При установке отметьте "Add Python to PATH"

2. Запустите приложение одним из способов:

   СПОСОБ 1 (Рекомендуемый):
   - Двойной клик на файл "start.bat"
   - Скрипт автоматически создаст виртуальное окружение,
     установит зависимости и запустит приложение

   СПОСОБ 2 (PowerShell):
   - Правый клик на "start.ps1" → "Запустить с помощью PowerShell"
   - Или откройте PowerShell в папке проекта и выполните: .\start.ps1

   СПОСОБ 3 (Ручной):
   - Откройте командную строку или PowerShell в папке проекта
   - Выполните: python run.py

3. После запуска откройте браузер и перейдите по адресу:
   http://127.0.0.1:8000/

4. Для доступа к административному интерфейсу:
   http://127.0.0.1:8000/admin/
   Логин: admin
   Пароль: admin123

РЕШЕНИЕ ПРОБЛЕМ:
============================================================

Если возникают ошибки:

1. Убедитесь, что Python установлен и добавлен в PATH
2. Проверьте, что порт 8000 не занят другим приложением
3. Попробуйте запустить: python manage.py runserver 8080
4. Для сброса базы данных удалите файл db.sqlite3 и запустите: python init_db.py
